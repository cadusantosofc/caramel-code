🏛️ Sobre o Projeto

O Caramel Code foi desenvolvido como projeto de TCC do curso Técnico em Desenvolvimento de Sistemas da ETEC Pedro D'Arcadia Neto.

Nosso objetivo é democratizar o ensino de programação, oferecendo uma plataforma gamificada e acessível para estudantes brasileiros.

TCC • Desenvolvimento de Sistemas • 2025

Tecnologias Utilizadas

React 18 TypeScript Tailwind CSS Framer Motion PWA LocalStorage Figma Design Responsive Design

Nossa Missão

"Tornar o aprendizado de programação divertido, acessível e eficaz para todos os brasileiros, independente do nível de conhecimento."
